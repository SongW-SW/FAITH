# FAITH

This is the code for the paper FAITH: Few-Shot Graph Classification with Hierarchical Task Graphs.


### Requirement:
torch==1.8.1



### Code Runing:
Run the command: 
`unzip data.zip`

Then run the command:
`python train.py`

To switch the dataset, modify the "datasets" variable in line 1192 of train.py.
